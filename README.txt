bbPress Topic and Reply Author Override

== Description ==

This plugin can help to change the authors of bbPress topics and replies from WordPress admin. Once installed it will add a metabox (Topic/Reply Author Override) on bbPress Topic and Reply screen.

If you want to know the codes behind this plugin then <a href="https://www.proy.info/bbpress-topic-and-reply-author-change-override/">Read more..</a>

== Installation ==

1. Upload the bbpress-topic-and-reply-author-override folder to the /wp-content/plugins/ directory
2. Activate the bbPress Topic and Reply Author Override plugin through the 'Plugins' menu in WordPress
3. Navigate to a topic/reply edit screen - look for Topic/Reply Author Override metabox on the right side.
